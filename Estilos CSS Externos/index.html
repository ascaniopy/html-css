Principais formatações

Negrito / Destaque

Nesta frase, tenho um termo em negrito usando a tag <B> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <STRONG> (semântica).

Itálico / Ênfase

Nesta frase, tenho um termo em itálico usando a tag <I> (não Semântica).

Nesta frase, tenho um termo em ênfase usando a tag <EM> (semântica).

Texto Marcado

Podemos criar também um texto marcado com a tag <MARK>.

Texto Grande e Texto pequeno

Estamos criando um texto grande, com a tag <BIG> e um texto pequeno, com a tag <SMALL>. nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, com a tag <DEL>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <INS>.

Existe também o sublinhado com a tag <U>. (não semântica).

Texto Sobescrito

Para inserir coisas do tipo x20 + 3, com a tag <SUP>.

Texto Subescrito

Para inserir coisas do tipo H2O, com a tag <SUB>.
